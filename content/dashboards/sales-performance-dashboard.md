## Dashboard Overview

This dashboard provides a comprehensive view of sales performance metrics including:

- **Revenue trends** over time
- **Regional breakdowns** with geographic mapping
- **Product category analysis** with drill-down capabilities
- **YoY comparison** metrics

### Key Features

- Interactive filters for date range and region
- Mobile-responsive design
- Real-time data refresh
- Export to PDF functionality

### Technologies Used

- Power BI Desktop
- DAX for calculations
- Power Query for data transformation
